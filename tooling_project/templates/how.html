How to use this app


You might be asking yourself, "Okay, this seems like a good tool to use as a new CS student, but how do I use it?"


Totally understandable, and as a new CS student, there are terms, tools, and technologies that are new to you.


Here is a list of items to start with, get familiar with, and use to your advantage.


  • First is git and Github. In the Tutorial Page, there are a number of links to help become more familiar with both.

  • Second, have a look at the tools that this app has you initialize and implement.

    • Sonarqube

    • Travis-CI doc for beginners

  • Third, because you're CS students, knowing the difference between proprietary and open-source is vital. Some questions to ask yourself -

    • What operating system am I using? Is it Microsoft Windows? Is it a Linux distro like Fedora or Ubuntu? Is it a Mac and using MacOS?

    • What is the difference between the OS's above? The companies that developed them?

    These are important questions to answer and can affect your choices of what operating system you would like to use and why.

  • Finally, this app was created as an open-source tool, and therefore, you can look at the Source Code and see how it was developed.

Good luck with your development!


- James Richardson
